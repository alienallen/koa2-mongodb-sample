# Restful API sample with Koa2 + MongoDB

> A node.js Restful API with Koa2 & MongoDB & winston & nodemon

## Build Setup


```bash
# clone the project
git clone 

# enter the project directory
cd koa2-mongodb-sample

# install dependency
npm install

# develop
npm run start
```
